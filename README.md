# Adapt-e
API Adapt-e

INSTALL

`composer install`

after

`php artisan key:generate`

create database and configure your database in file .env

`DB_DATABASE=database
DB_USERNAME=user
DB_PASSWORD=password`

after config .env file, run

`php artisan migrate`

install passport

`php artisan passport:install`

run api

`php artisan serve`
